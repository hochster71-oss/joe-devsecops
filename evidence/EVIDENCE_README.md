# J.O.E. DevSecOps Arsenal - Evidence Package

## Overview

This directory contains security and compliance evidence for continuous Authorization to Operate (cATO)
in accordance with DoD Enterprise DevSecOps Reference Design v2.0.

## Structure

```
evidence/
├── build/                    # Build verification evidence
│   └── build-manifest.json   # TypeScript, ESLint, build status
├── tests/
│   ├── unit/                 # Unit test coverage reports
│   ├── integration/          # Integration test results
│   └── e2e/                  # Playwright E2E test evidence
├── security/
│   ├── secrets/              # Secret scanning results (Gitleaks)
│   ├── sast/                 # Static analysis (Semgrep, ESLint)
│   ├── sca/                  # Dependency audit (npm audit)
│   ├── sbom/                 # Software Bill of Materials
│   │   ├── ASBOM.json        # Application Security BOM
│   │   ├── ABSMap.json       # Application Behavior Surface Map
│   │   └── cyclonedx-sbom.json # CycloneDX dependency SBOM
│   ├── container/            # Container image scanning
│   └── iac/                  # Infrastructure as Code scanning
└── remediation/              # Fix documentation by PR
    └── pr-{id}/              # Per-PR remediation evidence
```

## Compliance Frameworks

This evidence package supports assessment against:

- **NIST SP 800-53 Rev. 5** - Security and Privacy Controls
- **DoD STIG** - Security Technical Implementation Guides
- **CMMC Level 3** - Cybersecurity Maturity Model Certification
- **FedRAMP** - Federal Risk and Authorization Management Program

## Evidence Collection

### Automated (CI Pipeline)

The GitHub Actions workflow (`.github/workflows/devsecops.yml`) automatically collects:

1. **Build Evidence** - TypeScript, ESLint pass/fail
2. **Unit Test Coverage** - Vitest coverage reports
3. **SAST Results** - Semgrep SARIF output
4. **SCA Results** - npm audit JSON
5. **Secret Scan** - Gitleaks results
6. **SBOM** - CycloneDX dependency manifest
7. **E2E Results** - Playwright reports

### Manual Collection

Run locally:
```bash
# Full verification
npm run verify:full

# Individual scans
npm run typecheck          # TypeScript
npm run lint               # ESLint
npm run test:coverage      # Unit tests
npm run security:audit     # npm audit
npm run security:testid    # data-testid coverage
npm run security:sbom      # Generate CycloneDX SBOM
```

- Build evidence: 7 days
- Test evidence: 30 days
- Security scans: 90 days
- cATO package: 365 days
